# Path Finder
Codificador automatico de contraseñas a Hash para poder comparar un Hash con los hashes de las contraseñas de un diccionario hasta encontrar una coincidencia, la app ha sido desarrollada plenamente en Python y esta destinada a un uso ético.
- La applicación es un codificador automatico que genera y compara hashes instantaneamente, dependiendo del tipo de hash introducido (SHA256 o SHA512) generará uno u otro de forma automatica. 
Para el completo uso de la app se necesita de un Hash que se quiera buscar y de un diccionario SIN FORMATO de contraseñas o información, del modo que tenga en una fila información y salto de linea, información y salto de linea asi constantemente

Ejemplo de como tendría que estar el txt:
~~~
hola
hola123
hola1314
~~~
- Esta herramienta ha sido diseñada con el objetivo de poder comprender mejor el funcionamiento de la codificacíon hash y poder hacer ejercicios practicos con ello, además de posibles recuperaciones de contraseñas o busquedas en diccionarios para saber si son seguras las contraseñas que ya hemos establecido. _English version_
- Automatic password hash encoder to compare a hash with the hashes of passwords in a dictionary until a match is found. The app has been developed entirely in Python and is intended for ethical use.
- The application is an automatic encoder that instantly generates and compares hashes. Depending on the type of hash entered (SHA256 or SHA512), it will automatically generate one or the other. For full use of the app, you need a hash to search for and a dictionary of passwords or information in plain text, with each line containing information followed by a line break.

Example of how it should look the txt:
~~~
hello
